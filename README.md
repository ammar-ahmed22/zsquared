### Usage
#### Prerequisites
- Have `npm` or `yarn` installed!
- Have the server up and running, check the documentation here for instructions!

### Running locally
1. **Clone the repository:**
```sh
git clone https://google.ca
```

2. **Install dependencies:**
```sh
yarn # npm i
```

3. **Start the app:**
```sh
yarn start # npm start
```
